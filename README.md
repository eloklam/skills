# skills

A list of skills created by me for AI agents, published via `npx skills`.

## Skills List

### [university-solution-explainer](https://github.com/eloklam/university-solution-explainer)
- Let agents explain solutions from university STEM courses starting from your point of confusion.
- Polished for engineering students in Germany but universal.


### [siyuan-agent](https://github.com/eloklam/siyuan-agent)
- Standalone CLI for [SiYuan Notes](https://github.com/siyuan-note/siyuan) with direct HTTP API access.
- Allow agents to search, read, write and manage SiYuan elements easily. 
- Include logic for performance and speed by identifying task type.
